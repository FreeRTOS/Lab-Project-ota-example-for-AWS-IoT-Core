## coreOTA

### Usage

```
nix build
./result certificateFilePath privateKeyFilePath rootCAFilePath endpoint thingName
```

or

```
nix develop

cd build
cmake ..
make
./coreOTA certificateFilePath privateKeyFilePath rootCAFilePath endpoint thingName
```
